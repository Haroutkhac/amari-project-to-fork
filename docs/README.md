# Documentation

Project documentation and requirements.

## Files

- **projectDescription.md** - Original project requirements and specifications
- **googleform.txt** - List of fields to extract from shipment documents

## Field Requirements

The system extracts the following fields from shipment documents:

1. Bill of lading number
2. Container Number
3. Consignee Name
4. Consignee Address
5. Date of export
6. Date
7. Line Items Count
8. Average Gross Weight
9. Average Price
